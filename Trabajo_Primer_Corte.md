# 📌 Trabajo Primer Corte – Minería de Datos

Estimados estudiantes,

Para el primer corte deberán desarrollar un trabajo grupal aplicado de minería de datos, donde evidencien el entendimiento del proceso analítico desde la definición del problema hasta la interpretación de resultados.


## 🎯 Objetivo

Aplicar las fases iniciales del proceso CRISP-DM:

- Comprensión del problema
- Comprensión de los datos
- Preparación de los datos
- Análisis exploratorio (EDA)
- Interpretación de resultados

## 🧩 Actividad

Cada grupo debe seleccionar un dataset real (Kaggle, datos abiertos, gobierno, etc.) y desarrollar lo siguiente:

### 1️⃣ Comprensión del problema

No es solo describir, es pensar como ingeniero:

- ¿Qué problema están resolviendo?
- ¿Qué decisión se podría tomar con estos datos?
- ¿Quién usaría este análisis?
- ¿Por qué es importante?

🔍 Preguntas guía:

- ¿El problema es descriptivo, predictivo o explicativo?
- ¿Qué variable sería la más importante a analizar?
- ¿Qué riesgo hay si se interpreta mal este problema?

### 2️⃣ Comprensión de los datos

- Descripción del dataset
- Tipos de variables
- Valores faltantes
- Tamaño

🔍 Preguntas guía:

- ¿Qué variables parecen más relevantes? ¿Por qué?
- ¿Hay variables redundantes o inútiles?
- ¿Se evidencian sesgos en los datos?
- ¿Qué problemas de calidad detectan?

### 3️⃣ Limpieza y preparación

- Tratamiento de nulos
- Eliminación de duplicados
- Transformaciones

⚠️ OBLIGATORIO justificar TODO

🔍 Preguntas guía:

- ¿Qué pasa si NO limpian los datos?
- ¿Qué decisiones podrían alterar los resultados?
- ¿Están perdiendo información al limpiar?

### 4️⃣ EDA – Análisis Exploratorio

Deben incluir mínimo:

- Histogramas
- Boxplots
- Scatterplots
- Matriz de correlación

🔍 Preguntas guía:

- ¿Cómo se distribuyen los datos?
- ¿Existen outliers? ¿son errores o información valiosa?
- ¿Qué relaciones entre variables son evidentes?
- ¿Hay correlaciones fuertes? ¿tienen sentido lógico?

### 5️⃣ Interpretación

- ¿Qué patrones encontraron?
- ¿Qué variables influyen más?
- ¿Qué conclusiones reales se pueden sacar?
- ¿Qué decisiones podrían tomarse con estos resultados?

🔍 Preguntas guía:

- ¿Qué hallazgo es el más importante?
- ¿Qué resultado los sorprendió?
- ¿Qué hipótesis pueden plantear?
- ¿Qué harían como siguiente paso (modelo)?

## 📦 Entregable

- Informe en PDF
- Código en Python o R (opcional pero recomendado)
- Gráficas claras y legibles
- Interpretación nivel analista de datos
